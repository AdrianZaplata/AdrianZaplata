## Featured projects

### 🔎 [Job RAG](https://github.com/AdrianZaplata/job-rag)
Production RAG system over AI-engineer job postings — ingests postings, extracts structured skill data, scores each against a profile, and exposes the corpus through a LangGraph agent, FastAPI service, and MCP server.
- **RAG:** pgvector cosine search over `text-embedding-3-small`, section-based chunking, CPU cross-encoder reranking
- **Agent:** LangGraph ReAct with tool calling and SSE streaming
- **Structured extraction:** Instructor + Pydantic; atomic-skill decomposition lifted RAGAS context precision 0.60 → 0.67
- **Evaluation:** RAGAS golden dataset (18 queries, 4 metrics, measurable precision gain)
- **Production:** FastAPI with auth + rate limiting + SSE, Langfuse tracing, Docker Compose, GitHub Actions CI/CD, 139 tests

### 🎮 Ploop — *"Make My Game"*
Type a game idea, and ~15 seconds later a polished, mobile-friendly Phaser 3 arcade clip lives at a shareable link. Every clip is a self-contained mini-game generated by Claude.
- **Prompt engineering:** versioned system prompts, structured XML output with server-side extraction, sized for prompt caching
- **Model tiering:** Haiku default, Sonnet premium, single-pass repair loop on runtime failures
- **Eval-driven:** boot-rate as the north-star metric, JSON eval artifacts versioned in-repo
- **Safety:** sandboxed iframe on a separate origin, strict CSP, server-side content scanner
- **Tech:** TypeScript end-to-end, Next.js, React, Phaser 3, Anthropic Claude API, Supabase, Upstash Redis
