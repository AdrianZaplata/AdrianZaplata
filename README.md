## Featured projects

### 🔎 [Job RAG](https://github.com/AdrianZaplata/job-rag)
Production RAG system over AI-engineer job postings — ingests postings, extracts structured skill data, scores each against a profile, and exposes the corpus through a LangGraph agent, FastAPI service, and MCP server.
- **RAG:** pgvector cosine search over `text-embedding-3-small`, section-based chunking, CPU cross-encoder reranking
- **Agent:** LangGraph ReAct with tool calling and SSE streaming
- **Structured extraction:** Instructor + Pydantic; atomic-skill decomposition lifted RAGAS context precision 0.60 → 0.67
- **Evaluation:** RAGAS golden dataset (18 queries, 4 metrics, measurable precision gain)
- **Production:** FastAPI with auth + rate limiting + SSE, Langfuse tracing, Docker Compose, GitHub Actions CI/CD, 139 tests

### 🚲 [Relay](https://github.com/AdrianZaplata/relay)
Real-time telemetry platform for a connected e-bike fleet — an end-to-end vertical slice of a cloud-native micromobility system, from simulated pedelecs in the field to a live operator console.
- **Event-driven pipeline:** telemetry keyed by asset ID flows through Redpanda (Kafka API, 3 partitions) into an idempotent ingestor that upserts hot state into PostgreSQL
- **Operator console:** React + TypeScript dashboard with a live GPS fleet map (react-leaflet), backed by FastAPI — no polling
- **Lifecycle state machine** with transition validation (illegal transitions rejected with 409s), audit logging, hot/cold data split
- **Production-shaped ops:** ingestor and API deploy independently for separate scaling; Helm charts, ArgoCD GitOps, Terraform on STACKIT; ADRs + documented trade-offs; one-command demo via `make demo`
- **Tech:** Python, FastAPI, async SQLAlchemy/asyncpg, Redpanda, PostgreSQL, React, TypeScript, Vite, Docker, Kubernetes, Helm, Terraform, ArgoCD
